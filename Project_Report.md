# AETHERIS AI

## Project Overview

Aetheris AI is a space-themed FAQ chatbot developed using Python and JSON.

The chatbot answers frequently asked questions related to astronomy, planets, galaxies, black holes, and space exploration.

---

## Objectives

- Build a chatbot using NLP concepts.
- Provide instant answers to user queries.
- Create a beginner-friendly AI application.

---

## Technologies Used

- Python
- JSON

---

## Features

- FAQ-based chatbot
- Fast response system
- User-friendly interaction
- Expandable knowledge base

---

## Future Improvements

- GUI interface
- Machine learning integration

## Conclusion

The project successfully demonstrates chatbot development using Python and basic NLP concepts.
